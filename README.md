# 成長トラッカー

自己成長を記録・可視化するWebアプリ。

## 機能
- 5属性（行動・実績・能力・精神・知識）でポイント管理
- レーダーチャートでレベル内スコアを可視化
- レベルアップ条件: 全属性でそのレベル分のポイント獲得（Lv1→5pt、Lv2→6pt…）
- 超過分は次レベルへ繰越
- ×2ボタンで2点記録（1時間以上かかった行動など）
- 精神属性の記録でイシューを1つ消去
- データはlocalStorageに自動保存

## GitHub Pages へのデプロイ

1. このリポジトリをGitHubにpush
2. Settings → Pages → Source: GitHub Actions に設定
3. mainブランチにpushするたび自動デプロイ
